# Örnek Python Dosyaları

Bu dizinde Be My Code IDE ile oluşturulabilecek örnek Python dosyaları bulunmaktadır.

## Örnekler

### 1. basic_variables.py
Temel değişken tanımlama örnekleri

### 2. loops_demo.py
For ve while döngü örnekleri

### 3. functions_demo.py
Fonksiyon tanımlama örnekleri

### 4. calculator.py
Basit hesap makinesi

## Ses Komutları ile Oluşturma

Bu dosyaları ses komutları ile oluşturmak için:

1. Be My Code IDE'yi açın
2. `Ctrl+M` tuşuna basın
3. İlgili komutları sırayla söyleyin
4. Kod otomatik olarak oluşacaktır

Örnek:
- "string değişken adı isim"
- "integer değişken adı yaş"
- "for döngüsü yaz"
